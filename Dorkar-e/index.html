DORKAR-E

A stack of daily necessary apps

It is a collection of Apps consisting of Doctor Finder, ATM booth Locator and Restaurant Locator.
The app can be used for Bangladeshi marketplace. It makes sure that user can find a easy solution for daily usage in our complex life.

HOW IT WORKS

These are the following steps to use this application:
--- New user will sign up
--- After signing up for the first time the user will be able to sign in using his username and password
--- User will be able to find nearest doctor, restaurant, ATM booth by searching in the google map.
--- In the map the user will be able to see the shortest route of his destination and all other useful information
